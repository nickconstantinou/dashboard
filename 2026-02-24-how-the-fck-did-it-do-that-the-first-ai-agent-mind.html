How the f*ck did it do that? - The First AI Agent Mind-Blowing Moment

Summary

Peter Steinberger shares his experience with an early AI agent that completely astonished him with its capabilities. He opens by addressing the common skepticism that "there's no magic" in AI‚Äîit's just doing various tasks. But he argues that magic often comes from combining existing elements in new ways, using the example of iPhone scrolling: all the components existed before, but nobody had brought them together in that particular way until Apple made it feel obvious in hindsight.

The truly mind-blowing moment came when Peter was testing the AI agent and sent it an audio message. The agent displayed a typing indicator, even though it was supposedly only built for image support. When it responded successfully, Peter investigated how it had managed this feat. The agent had figured out that the file had no extension, determined it was an audio file (OOS format), used ffmpeg to convert it, found an OpenAI API key in the environment, and used curl to send it to OpenAI for transcription‚Äîall without being explicitly taught any of these steps.

Peter emphasizes that this demonstrates something profound: the agent has "world knowledge" and "creative problem-solving" capabilities. It chose the fastest path (using the API rather than downloading a local Whisper model), figured out file formats, and understood which tools to use. He connects this to programming skills, noting that being good at coding requires being good at general-purpose problem solving, and this skill appears to transfer to AI agents.

He then discusses the origins of his project (which became the foundation for Cursor), explaining it started as a WhatsApp relay but he moved it to Discord due to privacy concerns about sharing his phone number. The system had no security sandboxing at first‚Äîhe simply prompted the agent to only listen to him. People tried to hack it, but he kept developing openly, and that's when it really clicked for people who saw it. This was January, and it was his first time gaining attention from influencers.

    Main Takeaways

  • Magic comes from recombination: True innovation often isn't about inventing new components, but rather bringing existing elements together in novel ways that feel obvious only in hindsight.
  • AI agents can exhibit emergent problem-solving: Peter's agent independently figured out file conversions, API usage, and tool selection without explicit instructions‚Äîdemonstrating general-purpose reasoning abilities.
  • World knowledge enables smart decision-making: The agent chose the most efficient path (API over local model) based on contextual understanding, showing how AI can make nuanced decisions.
  • Early AI development required bold experimentation: Building in the open, without sandboxing or full security, allowed for rapid iteration and helped people truly understand what these systems could do.
  • Direct experience is crucial: Peter notes that you really need to experience AI agents to understand their capabilities‚Äîreading about them doesn't capture the same sense of wonder.
