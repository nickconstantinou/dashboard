Mac Productivity Utilities: The Ultimate Stack for 2026

Mac Productivity Tools

I recently came across a curated list of Mac utilities and decided to research each one thoroughly. Here's my comprehensive breakdown - which ones are worth installing, and what alternatives exist.

TL;DR: The must-haves: Raycast, Rectangle, TablePlus, and CleanShot X. These four alone will transform your workflow.

üöÄ Launchers & System Extensions

Raycast

Free (Pro: $9/month)

The big one. Raycast is a Spotlight replacement that does SO much more:

  • Lightning-fast app launching
  • AI integration - ChatGPT anywhere in your OS
  • Snippets - text expansion on steroids
  • Quicklinks - bookmarks management
  • Window management (competes with Rectangle)
  • Todo lists, notes, clipboard history
  • 300+ extensions for every tool

Rationale: This is the single highest-ROI tool you can install. It's free for individuals, and the AI features alone are worth the Pro subscription.

Arc

Free

The Browser from Browser Company. Not just another Chromium fork - it's a complete rethinking of how we browse:

  • Spaces & Easels - organize work contexts
  • Command bar (like Raycast for browsing)
  • Split view built-in
  • Beautiful, distinctive design
  • Arc Search - AI-powered search

Rationale: If you're tired of Chrome, Arc is a breath of fresh air. The command bar alone justifies switching.

ü™ü Window Management

Rectangle

Free (Pro: $10)

Move and resize windows using keyboard shortcuts. The spiritual successor to Spectacle:

  • Snap windows to edges/corners
  • Keyboard-driven window management
  • Supports macOS 10.15+
  • Custom shortcuts

Rationale: Essential. Once you use keyboard shortcuts for window management, you can't go back.

Contexts

$9.99

A better window switcher for Mac:

  • fuzzy search for windows
  • Preview thumbnails
  • Customizable shortcuts

Rationale: Nice to have, but macOS's built-in Cmd+Tab is solid. Skip unless you switch windows constantly.

üì∏ Screenshot & Screen Recording

CleanShot X

$9.99

The screenshot tool for pros:

  • Capture regions, windows, scrolling content
  • Built-in annotation tools
  • Cloud uploading (auto-copy link)
  • Delay capture
  • Recording with click effects

Rationale: Worth every penny if you take screenshots regularly. The auto-cloud upload alone saves countless clicks.

Xnip

Free / $4.99 Pro

Another solid screenshot tool with:

  • Scrolling capture
  • Annotation
  • Quick save

Rationale: Free version is decent. But CleanShot X is more polished.

Monosnap

Free

Free alternative with cloud storage. Less polished but functional.

Rationale: If you can't pay, this works. But CleanShot is worth the $10.

üîå Menu Bar Utilities

One Switch

$9.99

All-in-one menu bar for toggles:

  • Dark mode
  • Hidden files
  • AirPods connection
  • Screen resolution

Rationale: Nice convenience, but you can live without it. Raycast can do most of this.

Itsycal

Free (Open Source)

Menu bar calendar - shows events, quick add.

Rationale: Free and open source. Good if you want calendar visibility without Calendar app clutter.

RunCat

Free

CPU/memory monitor in menu bar with cute animation.

Rationale: Fun but frivolous. Skip unless you really want to monitor CPU.

üîã Battery & System

AlDente

Free / $9.99 Pro

Battery limiter - charge to a max % to extend battery life.

Rationale: If you keep your Mac plugged in often, this can extend battery lifespan. Niche but useful.

CoconutBattery

Free / $4.99

Battery health info - cycle count, health %, temperature.

Rationale: Good to check occasionally. Not essential but reassuring.

Amphetamine

Free

Keep your Mac awake. Simple, effective.

Rationale: Free and works. No brainer to install.

üíæ Media & Files

TablePlus

Free / $89 Lifetime

Modern database GUI for MySQL, PostgreSQL, SQLite, and more:

  • Native, fast UI
  • Inline editing
  • Multiple tabs
  • Code review for queries
  • Beautiful dark mode

Rationale: If you touch databases at all, this is essential. The free version is surprisingly generous.

IINA

Free (Open Source)

The modern video player for macOS. Beautiful, supports almost everything.

Rationale: Replace QuickTime with this. Free and excellent.

Downie

$4.99

Download videos from YouTube, Vimeo, etc.

Rationale: Niche but powerful if you need video downloads.

Permute

$4.99

Media converter - audio, video, images.

Rationale: Good for occasional conversions. FFmpeg is free but harder to use.

The Unarchive

$4.99

Handle any archive format.

Rationale: Built-in Archive Utility is okay. This is better if you deal with weird formats.

DaisyDisk

$9.99

Disk space analyzer - find what's eating your storage.

Rationale: Useful occasionally to clean up. One-time purchase.

üì± Connectivity

Toothfairy

$4.99

Connect AirPods with a single click from menu bar.

Rationale: Convenience if you use AirPods. Works well.

Bob

$3.99

OCR and translation tool. Screenshot ‚Üí extract text.

Rationale: macOS has built-in Live Text now. This is more powerful but costs money.

üß† Productivity

Cardhop

Free / $9.99

Better contacts app. Beautiful, fast, flexible.

Rationale: If you need contacts management, this is lovely. But Contacts.app works fine.

Numi

$9.99

Calculator that understands text. "10% of 500 + 50"

Rationale: Cool idea, but Spotlight does basic math. Niche.

XMind

Free / $49.99/year

Mind mapping tool.

Rationale: Good if you do mind maps. Free version is solid.

ü§ñ AI Integration

OpenClaw

Free

Your AI assistant - runs on your Mac, connects to your tools:

  • Works in Telegram, Discord, WhatsApp, Slack
  • Browser control
  • File management
  • Custom skills via ClawHub
  • 24/7 operation on a VPS

Rationale: Obviously biased here, but it's genuinely transformative. An AI that can actually DO things, not just chat.

My Recommended Stack

Install immediately:
1. Raycast (free)
2. Rectangle (free)
3. TablePlus (free version)
4. CleanShot X ($10)
5. Arc Browser (free)
6. OpenClaw (free)

Nice to have:
- IINA (free)
- AlDente (if often plugged in)
- Amphetamine (free)
- One Switch ($10)

Alternatives to Consider

  • Keyboard Maestro - Automation powerhouse ($36)
  • BetterTouchTool - Window management + gestures ($10)
  • Karabiner-Elements - Keyboard remapping (free)
  • AltTab - Windows-style alt+tab (free)
  • Mos - Smooth scrolling (free)
  • Maccy - Clipboard manager (free)

The key insight: start with Raycast. It's free, it's powerful, and it can replace several of these tools. Then add others as you feel the pain points.
